# Conspiracy
A deductive game along the lines of Avalon and The Resistance.

## Getting Started

### Server

- Start a clojure REPL
- `(in-ns 'conspiracy.server)`
- `(start sys)`

### Client

- From a terminal: `shadow-cljs watch app`
- Open `public/index.html` in a browser
